## プロンプト本文

```
フロントエンドからインフラまでを縦に貫くエンドツーエンドのリクエストフロー図を作成してください。

【内容】
- 3つの水平レイヤーを縦に積む:
  - 上段「フロントエンド」: ユーザーがボタンをクリック → コンポーネント → API 関数呼び出し
  - 中段「ネットワーク・インフラ」: CloudFront → ALB → ECS コンテナ
  - 下段「バックエンド」: Controller → UseCase → Model → データベース
- 1本の明るい線（リクエスト）が上段のユーザー操作から下段のデータベースまで縦に貫通する
- 同じ線がレスポンスとして下から上に戻り、UI が更新される
- 全レイヤーが1つの機能でつながっていることを強調

【スタイル】
- フラットデザイン・インフォグラフィック風
- 白背景、青（#3B82F6）とオレンジ（#F97316）のアクセントカラー
- クリーンな線画、ラベルは日本語
- 16:9 横長レイアウト
```
