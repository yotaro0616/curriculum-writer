## プロンプト本文

```
JavaScript の Event Loop（イベントループ）のメンタルモデル図を作成してください。

【内容】
- 中央に円形の回転ベルトコンベア（イベントループ）。1人の作業員（JavaScript のシングルスレッド）が処理ステーションに立ち、ベルト上のアイテムを1つずつ処理する
- 左側に2つの待機エリア: 上段「マイクロタスクキュー」（Promise コールバック、優先バッジ付き）、下段「タスクキュー」（setTimeout コールバック、通常チケット）。マイクロタスクが常に先にベルトに乗る
- 右側に「Web APIs」の作業場。バックグラウンドワーカーが fetch() や setTimeout の長時間処理を担当し、完了したらキューに結果を戻す

【スタイル】
- フラットデザイン・インフォグラフィック風
- 白背景、青（#3B82F6）とオレンジ（#F97316）のアクセントカラー
- クリーンな線画、ラベルは日本語
- 16:9 横長レイアウト
```
