```
Next.js の Server Components と Client Components の境界を示すメンタルモデル図を作成してください。

【内容】
- コンポーネントツリーを縦に描く。上部のコンポーネント（page.tsx, layout.tsx, データ取得コンポーネント）は青色で、雲のアイコン「サーバー（Node.js）」の中にある
- 中央に水平の破線「use client 境界線」が引かれている
- 下部のコンポーネント（ボタン、フォーム、インタラクティブUI）はオレンジ色で、ブラウザウィンドウのアイコンの中にある
- Props のデータが上（サーバー）から下（クライアント）へ矢印で流れている

【スタイル】
- フラットデザイン・インフォグラフィック風
- 白背景、青（#3B82F6）とオレンジ（#F97316）のアクセントカラー
- クリーンな線画、ラベルは日本語
- 16:9 横長レイアウト
```
