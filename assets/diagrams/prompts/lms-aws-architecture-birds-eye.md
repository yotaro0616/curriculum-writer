## プロンプト本文

```
LMS の AWS アーキテクチャ全体像を「街の鳥瞰図」として描いてください。

【内容】
- アイソメトリック（斜め上から見下ろす）視点の街並み
- 入口: CloudFront（料金所ゲート）。ユーザーのリクエストがここから入る
- 交通整理: ALB（交差点の交通整理員）。リクエストを振り分ける
- メインビル群: ECS Fargate コンテナ（Nginx + Laravel のビル）が2〜3棟並ぶ
- 倉庫: S3（大きな倉庫）。静的ファイルを保管
- 金庫: RDS Aurora（地下金庫のある銀行）。データベース
- ロッカー: DynamoDB（ロッカー施設）。セッション・キャッシュ
- 上部にユーザーのアイコンが複数並び、CloudFront に向かってリクエストを送っている

【スタイル】
- フラットデザイン・インフォグラフィック風のアイソメトリック
- 白背景、青（#3B82F6）とオレンジ（#F97316）のアクセントカラー
- クリーンな線画、ラベルは日本語
- 16:9 横長レイアウト
```
