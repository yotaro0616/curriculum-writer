## 用語テーブル

| 表記 | 備考 |
|---|---|
| デプロイ | 「デプロイメント」ではなく「デプロイ」で統一 |
| Vercel | カタカナ表記（ヴァーセル）は使わない。常に英語表記 |
| DNS | 初出時に正式名称（Domain Name System）を併記 |
| HTTPS | 初出時に HTTP との違いを簡潔に説明 |
| CDN | 初出時に正式名称（Content Delivery Network）を併記 |
| SPA | 初出時に正式名称（Single Page Application）を併記 |
| PaaS / IaaS | 初出時に正式名称を併記 |
